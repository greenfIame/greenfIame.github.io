This is a wolfenstien 3d remake in javascript using the HTML5 canvas and the p5js library.
			 Controls:

			 [W, A, S, D]		Player movement 
			 [MOUSE]			Used to look around
			 [LEFT CLICK]		Shoot
			 [SHIFT]			Zoom scoped weapon
			 [O]				Open/Close options

			 If the mouse is supported the mouse can be used to aim and click to fire,
			 should the browser not support this feature the left and right arrow keys
			 are to be used to aim and space to fire.
